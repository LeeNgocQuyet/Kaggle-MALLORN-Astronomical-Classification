# Contributor Code of Conduct

Be respectful and considerate. Contributions are welcome from everyone, regardless of background.

Expected behavior:
- Be professional and courteous in communication.
- Use inclusive language.
- Give constructive feedback and be open to suggestions.

Unacceptable behavior:
- Harassment, discriminatory language, or personal attacks.

If you witness or experience unacceptable behavior, please open an issue or contact the repository maintainers.
